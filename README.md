# Parking :: Parking Finder in flutter 


## Steup
ex Git :
run this commanda in Terminal
~~~
flutter clean
flutter pub get 
~~~
## Using ::
- dart
- Flutter Widget
- State Management (Bloc & Cubit)
- Firebase & Cloud FireStore
